# Command 07: Review Sign-Up UI

## Goal

Review the sign-up page before final checks.

## Checklist

- Correct route file is used.
- There is no duplicate register route.
- Uses TanStack Router `Link`.
- Uses shadcn components.
- Uses project theme classes.
- No hardcoded brand colors in TSX.
- No external image URLs.
- No Next.js imports.
- No `.env` access.
- Form is UI-only.
- Mobile layout works.
- Desktop layout fits without page scroll at the target auth viewport.
- Accessibility basics are covered.

Fix all critical issues before proceeding.
