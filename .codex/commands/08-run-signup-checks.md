# Command 08: Run Sign-Up Checks

## Run

```bash
nvm use
npm run format
npm run check
npm run build
```

## Manual QA

Check:

```text
/auth/register
```

Confirm:

- Header logo appears.
- `Sign in` link appears.
- Left marketing panel appears on desktop.
- Sign-up card appears on the right.
- Form is usable on mobile.
- All inputs are labeled.
- Terms checkbox is present.
- Continue with Google placeholder is present.
- No backend auth was added.
