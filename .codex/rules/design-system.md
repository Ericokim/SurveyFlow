# SurveyFlow Design System Rules

- Primary action: `bg-primary text-primary-foreground`.
- Text: `text-foreground`.
- Secondary text: `text-muted-foreground`.
- Cards: `bg-card border-border`.
- Use subtle shadows.
- Avoid nested cards.
- Avoid hardcoded brand hex colors in TSX.
- Keep radius and spacing consistent with shadcn.
