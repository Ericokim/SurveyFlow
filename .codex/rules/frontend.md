# SurveyFlow Frontend Rules

- Use React function components.
- Use TypeScript without `any`.
- Use TanStack Router `Link` for internal navigation.
- Use shadcn/ui primitives before custom controls.
- Use Tailwind theme classes and `cn()` for conditional classes.
- Use `@/` imports.
- Do not use Next.js imports.
- Do not use external image URLs for core product UI.
- Keep route files thin.
